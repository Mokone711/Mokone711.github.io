# 🚴 Cyclistic Case Study: How Does a Bike-Share Navigate Speedy Success?

## 📌 Overview

This case study analyzes how Cyclistic, a bike-share company, can increase annual memberships by converting casual riders into members.

## 🎯 Business Task

Identify key differences between casual riders and annual members and provide actionable recommendations to increase memberships.

## 📊 Data Source

Public bike-share trip data (Divvy dataset).

## 🛠 Tools Used

* SQL (Data Cleaning & Analysis)
* R (ggplot2 for visualization)
* Excel (Quick checks)

## 🔍 Data Cleaning Process

* Removed duplicates
* Handled missing values
* Standardized date formats
* Created new columns (ride_length, day_of_week,data_type)

## 📈 Analysis

Key questions explored(ASK):

* How do annual members and casual riders use Cyclistic bikes differently?
* Why would casual riders buy Cyclistic annual memberships?
* How can Cyclistic use didgital ,media to influence casual riders to become members?

## 💡 Key Insights

* Casual riders take longer rides on average
* Members use bikes more frequently on weekdays
* Casual riders are more active on weekends and for leisure

## ✅ Recommendations

1. Offer weekend promotions targeting casual riders
2. Introduce discounted membership trials
3. Create marketing campaigns focused on leisure riders
